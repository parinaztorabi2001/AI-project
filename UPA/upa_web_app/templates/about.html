About UPA System

What is UPA?

Unsupervised Prediction Alignment (UPA) is an innovative algorithm designed to automatically correct performance drift in medical image classification systems caused by changes in data acquisition.

This system is based on the research paper published in Nature Communications (2023) 14:6608 titled "Automatic correction of performance drift under acquisition shift in medical image classification".

How It Works

1. Data Acquisition

Medical images are collected from different scanners and protocols

2. Drift Detection

System detects performance drift caused by acquisition changes

3. UPA Alignment

Automatic alignment without needing new labels

4. Performance Recovery

Restores original sensitivity/specificity balance

Key Features

  • Unsupervised: No need for new labeled data
  • Automatic: Continuous monitoring and correction
  • Real-time: Immediate response to acquisition changes
  • Clinically Relevant: Preserves sensitivity/specificity balance
  • Scalable: Works with multiple scanners and protocols

Dataset Information

This system uses the Breast Cancer Wisconsin (Diagnostic) Dataset which contains 569 samples with 30 features computed from digitized images of fine needle aspirate (FNA) of breast masses.

Total Samples: 569
Benign Cases: 357 (62.7%)
Malignant Cases: 212 (37.3%)
Features: 30 numerical features

Technical Implementation

Backend: Python, Flask, Scikit-learn, NumPy, Pandas

Frontend: HTML, CSS, JavaScript, Chart.js

Algorithm: Random Forest Classifier with UPA correction

Visualization: Matplotlib for performance metrics

Clinical Relevance

UPA is particularly valuable in medical imaging where:

  • Hospitals upgrade or replace imaging equipment
  • Different sites use different scanner manufacturers
  • Software updates change image processing algorithms
  • Ensuring consistent diagnostic performance across sites
